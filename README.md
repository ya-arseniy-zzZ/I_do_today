**Personalized Task Tracker** — это веб-приложение для управления задачами, которое помогает пользователю планировать **реалистичный объём задач на день**, избегая перегрузки и фокусируясь только на действительно важных задачах.

Главная идея — не просто хранить список задач, а **рассчитывать реальную пропускную способность дня** и предлагать оптимальный набор задач.

Основная цель проекта:

> Помочь человеку спланировать столько дел, сколько реально он сможет сделать за день, а также сфокусироваться только на том, что действительно важно. 

---

# Problem

Большинство таск-трекеров (Todoist, Notion, Jira, Trello):

* позволяют создавать **неограниченное количество задач**
* не учитывают **реальное время пользователя**
* приводят к **перегрузке и хаосу**

В результате пользователь:

* планирует больше, чем может сделать
* теряет фокус
* чувствует постоянный backlog

Этот проект решает проблему через:

* расчёт **реального доступного времени**
* подбор **оптимального набора задач**
* фокус на **ежедневном execution**

# Core Concept

Главная концепция — **умное планирование дня**.

Приложение:

1. Получает доступное время пользователя
2. Анализирует задачи
3. Предлагает **реалистичный набор задач на день**

Для этого используются два метода:

## 1️⃣ Метод 1-3-5

Техника тайм-менеджмента:

* **1 большая задача** (3+ часа)
* **3 средние задачи** (1–2 часа)
* **5 маленьких задач** (15–30 минут)

Алгоритм автоматически заполняет день задачами на основе доступного времени.

# Core Features

## 1. Task Management

Пользователь может:

* создавать задачи
* назначать задачи проектам
* оценивать задачи
* ранжировать порядок выполнения

Это базовый функционал MVP. 

---

## 2. Daily Capacity Calculation

Система рассчитывает **реальную пропускную способность дня** на основе:

* доступного рабочего окна
* встреч из календаря
* оценки задач
* исторических данных пользователя

---

## 3. Calendar Integration

Интеграция с:

* Google Calendar
* Apple Calendar

Задачи планируются **с учётом встреч**. 

---

## 4. Focus List

Система формирует **фокус-лист задач на сегодня**.

В него попадают:

* наиболее приоритетные задачи
* задачи, которые реально успеть сделать

---

## 5. Pomodoro Tracking

Каждая задача может отслеживаться через **Pomodoro-таймер**.

Это позволяет:

* анализировать реальное время задач
* улучшать точность планирования
